- Answer to the question 

1️⃣ What is the difference between var, let, and const?
- Answer
     - var: This is the old way to declare variables. redeclare the same variable name multiple times, and it is function-scoped.
     - let: This is the modern way to declare variables. redeclare it in the same scope, but you can change its value later.
     - const: This is for fixed values. Once you declare it, the value cannot be changed or reassigned.

2️⃣ What is the spread operator (...)?
- Answer
     - The spread operator consists of three dots (...) used to copy all elements from an array or object into another.
     - It is the easiest way to combine multiple arrays or clone an object. 

3️⃣ What is the difference between map(), filter(), and forEach()?
- Answer
     - map(): It runs a function on every element and returns a new array with the results.
     - filter(): It checks a condition for each element and returns a new array containing only the items that pass the test.
     - forEach(): It simply loops through the data to perform an action (like console log), but it does not return a new array.

4️⃣ What is an arrow function?
- Answer
     - An arrow function is a shorter and more modern way to write functions in JavaScript.
     - You don't need to write the function keyword; instead, you use a fat arrow (=>).

5️⃣ What are template literals?
- Answer
     - Template literals allow you to write strings using backticks (`) instead of quotes. 
     - They make it easy to insert variables directly into a string using ${} and allow for multi-line text.
